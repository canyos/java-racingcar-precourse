피드백
1. 커밋 시에 어떠한 것을 했는지 앞에 적어주시기 바랍니다.
2. 함수를 기능별로 나누는 것이 좋아보입니다.결과를 출력하는 메소드 안에 게임이 진행되는 로직이 있는데 의존도가 높아져 바람직하진 않습니다.
3. 클래스 별로 구분을 하여 MVC 패턴으로 작성하지 않았습니다.
4. oneStep과 같은 네이밍은 해당 메소드의 동작을 추측할 수 없습니다.
